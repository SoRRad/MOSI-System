# Load from GitHub
library(jsonlite); library(tidyverse)

url <- "https://raw.githubusercontent.com/owner/mosi-system/main/data/cases.json"
cases <- fromJSON(url) |> filter(!isTest)

# Agreement rates by stage
cases |> group_by(stage) |> summarise(n=n(), proc_agree=mean(procAgreement=="Agree")*100)
How It Works
Architecture and setup guide for the MOSI platform.
⚠Jekyll Build Error? Create a file named .nojekyll (empty file, exact name including the dot) in the root of your repository. This stops GitHub Pages from running Jekyll.
ℹWorkflow File Location: The submit_case.yml file must be at .github/workflows/submit_case.yml (with the dot at the start) for GitHub Actions to recognize it.
