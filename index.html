Methodology Notes

Definitions and manuscript-aligned research results used by this static calculator.

MOSI-S

  • M1 BMI 30.0-34.9; M2 35.0-39.9; M3 40.0-44.9; M4 45.0-49.9; M5 BMI ≥50.0.
  • O score adds one point each for hypertension, dysglycemia, obstructive sleep apnea, and hyperlipidemia.
  • Si3 maps to Stage IV. Si2 maps to Stage III. Si1 maps to Stage III only with M ≥4 or O ≥3.

MOSI-T

  • Tier A target ≥7.5% TWL.
  • Tier B target ≥15% TWL.
  • Tier C target ≥25% TWL.
  • Tier D target ≥30% TWL.

MOSI-D

  • Displays research decision-support estimates for SG and RYGB.
  • Shows available v5.1 values at 12 and 60 months.
  • Unavailable exact threshold values display as N/A.

Study Numbers

  • Staged cohort: n = 3,192.
  • Primary 12-month analytic cohort: n = 2,939.
  • Primary endpoint: TWL ≥20% at 12 months.
  • MOSI-T distribution: A 248; B 654; C 1,549; D 488.
